Where The Web Was Born

Tim Berners-Lee,a British scientist,invented the
World Wild Web (WWW) in 1989, While Working at CERN.
The Web was originally conceived and developed to meet
the demand for automated information-sharing
between scientists in universities and institutes around the world.

Tim Berners-Lee,pictured at CERN (image:CERN)

How the web begun

The first page of Tim Berners-Lee's proposal for the World wide web,
written in March 1989

Tim Berners-Lee wrote the first proposal for the World wide Web
In March 1989 and his second proposal in May 1990.
Together with Belgian systems engineer Robert Cailliau,
this was formalised as a management proposal in November 1990.

This outlined the principal concepts and it defined important
terms behind the Web.The document described a"hypertext project" called
WorldWideWeb in which a "web" of "hypertext documents" could be viewed by "browsers".

By the end of 1990,Tim Berners-Lee had the first Web server and
browser up and running at CERN,demonstrating hid ideas. He developed the code
for his Web server on a NeXT computer. To prevent it being
accidentally switched off, the computer had a hand-written label in red link:
"This machine is a server.DO NOT POWER IT DOWN!!"
